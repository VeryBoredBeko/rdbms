# План уроков: Базы данных

## Урок 1: Основные понятия. Инструменты для работы с базами данных

### Материалы:

1. **Кузнецов, Базы данных**
2. Файл заданий: `task.md` с вопросами

### Инструкция для студентов:

1. Прочитайте назначенные главы из книги Кузнецова.
2. Выполните задания и ответьте на вопросы в файле `task.md` на основе прочитанного материала.
3. Сдайте выполненную работу в формате **.doc** или **.pdf**.

---

## Урок 2: Реляционная модель. Модель данных

### Материалы:

1. **Database Systems: The Complete Book (DS:CB)** — Hector Garcia-Molina, Jeff Ullman, Jennifer Widom.

### Инструкция для студентов:

1. Прочитайте выбранные главы из DS:CB.
2. Выполните упражнения, обозначенные как `exercise`, из DS:CB.
3. Сдайте выполненную работу в формате **.doc** или **.pdf**.

---

### Примечания для студентов:

* Каждый урок направлен на ваше понимание материала и применение его на практике через выполнение заданий.
* Обязательно предоставьте выполненные задания из `task.md` и упражнения из DS:CB для проверки.
