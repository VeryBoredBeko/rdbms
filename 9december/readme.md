# Задание на сегодня — SQL Academy

**Ссылка на тренажёр:** [https://sql-academy.org/ru/trainer](https://sql-academy.org/ru/trainer)

---

## Инструкция

- Выполнить **15 лёгких** и **25 средних** задач.  
- Для каждой задачи кратко записать **суть выполненной задачи**.  
- Сделать **скриншот успешного выполнения** каждой задачи.  
- Все скриншоты и записи о сути задач собрать в одном документе (PDF).  

---

**Цель задания:**  
Закрепить практические навыки работы с SQL, научиться формулировать суть задачи и фиксировать результат.
